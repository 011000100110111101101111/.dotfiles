## Themes

__Each theme has two versions. The desktop version (multiple monitors) and the laptop version. Most share the same configurations, which are located in Shared. They also both implement the same stuff from System. The differences between laptop and desktop are in Specific. This is usually customization things that need to be different on the desktop to fit the screen compared to the smaller laptop. Currently, waybar and SOME of the hypr configs are specific. To continue the hypr tangent, there are multiple linked hyprland configuration files. Most are located in shared, however some specific nvidia and startup stuff are located in specific. FYI, the specific hypr config is the base config, the ones in shared are sourced when it starts.__
